UFO Sightings Forecasting: เจาะสถิติและทำนายอนาคต UFO
โปรเจคนี้คือการนำข้อมูลการพบเห็น UFO ทั่วโลกในอดีตมาวิเคราะห์เพื่อหาคำตอบว่า "การรายงานพบ UFO มีรูปแบบที่แน่นอนไหม?" และเราจะพยากรณ์จำนวนการพบเห็นในเดือนถัดๆ ไปได้แม่นยำแค่ไหน โดยใช้เทคนิคทาง Data Science ครับ

- ทำไมโปรเจคนี้ถึงน่าสนใจ? (Problem Definition)
ข้อมูล UFO มักถูกมองว่าเป็นเรื่องบังเอิญหรือเหตุการณ์สุ่ม แต่ถ้าเราลองดูสถิติย้อนหลัง เราจะพบว่ามันมี "รอบเวลา" และ "แนวโน้ม" ที่ชัดเจน การสร้างโมเดลพยากรณ์จะช่วยให้เราเข้าใจพฤติกรรมการรายงานข้อมูล และแยกแยะได้ว่าช่วงไหนที่มีความผิดปกติเกิดขึ้นจริง

- เราเจออะไรในข้อมูลบ้าง? (Insights from EDA)
จากการสำรวจข้อมูล (Exploratory Data Analysis) เราพบจุดที่น่าสนใจดังนี้:

ทำความสะอาดข้อมูล (Data Cleaning): ข้อมูลดิบมีปัญหาเยอะมาก เช่น เวลาที่เขียนว่า 24:00 (คอมพิวเตอร์ไม่เข้าใจ) หรือพิกัดที่มีตัวอักษรปนมา เราได้ซ่อมแซมข้อมูลเหล่านี้ให้พร้อมใช้งาน

การจัดการค่าว่าง (Missing Values): ในส่วนของรูปทรง (Shape) ที่หายไป เราเลือกเติมคำว่า unknown แทนการลบทิ้ง เพื่อรักษาจำนวนนับในแต่ละเดือนให้แม่นยำที่สุด

ช่วงเวลายอดฮิต (Seasonality): UFO มักถูกรายงานมากที่สุดในเดือน "กรกฎาคม" ของทุกปี และช่วงเวลาที่พบมากที่สุดคือ "21:00 - 23:00" ซึ่งสัมพันธ์กับช่วงเวลาพักผ่อนของผู้คน

พื้นที่ (Location): ข้อมูลส่วนใหญ่กระจุกตัวหนาแน่นในสหรัฐอเมริกาและยุโรป

- การสร้างและเปรียบเทียบโมเดล
เราใช้การสรุปยอดรายเดือน และนำโมเดลมาประเมินผลแข่งกัน:

Linear Regression: เป็นตัวแทนของโมเดลเส้นตรง ดูแนวโน้มการเติบโต

Random Forest & Gradient Boosting: โมเดลระดับสูงที่ช่วยจับความสัมพันธ์ที่ซับซ้อน

Prophet: โมเดลพยากรณ์ที่เก่งเรื่องการจัดการข้อมูลที่มี "ฤดูกาล" (Seasonality)

- วิธีใช้งาน Web App
เข้าไปที่ URL ของ Streamlit App (ใส่ลิงก์ของคุณที่นี่)

หน้าเว็บจะแสดงกราฟวิเคราะห์ข้อมูลในอดีต (EDA)

คุณสามารถดูผลการพยากรณ์จำนวนการพบเห็น UFO ในอนาคตที่โมเดลคำนวณไว้ให้ได้ทันที
